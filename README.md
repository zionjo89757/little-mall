# littlemall
SpringBoot SpringCloud SpringCloud Alibaba

## 主要组件
- Nginx: 动静分离、负载均衡给上流服务器（SpringCloud gatewway）
- Gatewway: 解析Host域名，根据各域名地址负载均衡转发给相应的微服务
- Nacos: 配置中心、注册中心
- OpenFeign: RPC，微服务之间的远程调用
- Sentinel: 服务熔断、服务降级、服务限流
- ElasticSearch: 详情检索
- Redis: 缓存、分布式Session、分布式锁
- MySQL: 数据存储
